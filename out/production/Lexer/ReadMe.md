2.正规文法转换为正规表达式
2.1 左右线性文法互换

split["|"]

在一个for循环里进行遍历，遍历内部再嵌套一个遍历，外部实现把形如S::=aS|A转变为S=a*A,内部实现若产生式右边有非终结符，把该非终结符用它的右侧替换，循环到所有产生式右侧都只有终结符是结束
